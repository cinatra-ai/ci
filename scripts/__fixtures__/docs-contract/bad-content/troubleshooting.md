# WordPress troubleshooting

## Connection fails

- **Symptom:** "Could not connect" on Connect.
- **Cause:** Invalid or expired API key.
- **Fix:** Re-enter the key from your Cinatra dashboard.
- **Diagnostics:** Check the plugin debug log.
- **Escalation:** [Contact support](https://docs.cinatra.ai/resources/support/).
